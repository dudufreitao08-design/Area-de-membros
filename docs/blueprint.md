# **App Name**: Aether Academy

## Core Features:

- Login Screen: Authentication via email and password using Firebase Authentication.
- Registration Screen: User registration with email and password using Firebase Authentication.
- Password Recovery Screen: Password recovery functionality using Firebase Authentication.
- Dashboard: Internal panel with 5 modules (cards or lists with icons/images).
- Login Verification: Protected routing to ensure only logged-in users can access the dashboard.
- User Progress Tracking: Store user progress (completed modules) in Firestore.

## Style Guidelines:

- Primary color: Deep Night Blue (#0D1B2A) for tranquility and confidence.
- Secondary color: Serene Blue (#1B263B) for backgrounds and secondary sections.
- Text color: Light Neutral Gray (#E0E1DD) for soft contrast on texts and backgrounds.
- Support color: Relaxing Blue (#415A77) for secondary buttons, boxes, or titles.
- Detail color: Nebulous Blue (#778DA9) for subtle details and soft backgrounds.
- Accent color: Soft Gold (#D4AF37) used sparingly for primary CTAs, icons, or badges to convey exclusivity.
- Font: 'Poppins', a modern sans-serif font for headers and body, which provides a contemporary, precise feel.
- Smooth page transition animations for navigation.
- Dark mode with soft gradients and Tailwind CSS with Backdrop Blur, inspired by Discord's design.
